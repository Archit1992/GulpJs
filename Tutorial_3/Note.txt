Ref : https://css-tricks.com/gulp-for-beginners/
 
Step 1 : npm install gulp -g	// install gulpjs globally. 
step 2 : npm init		// initialize the node project.
step 3 : package.json file is created.
step 4 : npm install gulp --save-dev	// add gulp dependencies into the package.json file.
step 5 : npm install gulp-sass --save-dev	// add dependency.
step 6 : npm install gulp-concat --save-dev	// add dependency.
step 7 : npm install gulp-header --save-dev	// add dependency.
step 8 : npm install --save-dev jshint gulp-jshint 	// add dependency.
step 9 : npm install gulp-minify --save-dev	// add dependency.
step 10 : npm install gulp-rename --save-dev	// add dependency.
step 11 : npm install gulp-uglify --save-dev	// add dependency.

step 12 : run : gulp scripts
